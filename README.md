## FAQ

**Welche Dateiformate werden unterstützt?**
> Alle gängigen Formate: PDF, DOC/DOCX, XLS/XLSX, PPT/PPTX, JPG, PNG, GIF, ZIP, RAR, TXT, CSV, MP4 und AVI.

**Ist die Extension mit Hyvä kompatibel?**
> Ja — STM Product Attachment Pro ist nativ für Hyvä Theme optimiert und nutzt Alpine.js für die Inline-Vorschau und Modals.

**Wie funktioniert die automatische Zuweisung?**
> Regeln basierend auf Produkt-Kategorien, Attribut-Sets oder einzelnen Produktattributen (z.B. Hersteller, Material). Die Engine verteilt Attachments dann automatisch an alle passenden Produkte — per Apply-on-Save oder über einen konfigurierbaren Cron-Job.

**Kann ich Downloads auf bestimmte Kundengruppen beschränken?**
> Ja. Sie können Attachments auf bestimmte Kundengruppen einschränken und den Gastzugang separat steuern.

## Installation

> Composer-Installation funktioniert nach Kauf über den ausgelieferten Auth-Token / das ZIP-Paket. Eine öffentliche Packagist-Distribution ist für Q3/2026 geplant.

```bash
# Nach Kauf:
composer require storetown/module-attachment-pro
bin/magento module:enable Storetown_AttachmentPro
bin/magento setup:upgrade
bin/magento setup:di:compile
bin/magento setup:static-content:deploy
bin/magento cache:flush
```

Alle Einstellungen anschließend unter **Stores → Configuration → STM → Attachment Pro**.
